Т.к. с Angular не работал, поэтому ваш проект не запускал. Но на моем проекте скрипт работаетт.

Добавил в проект скрипт script.js, который сравнивает {{item.name}} с ключом в файле title.json (его тоже добавил)
при наведении мыши на SSL. Перебираются все ключи и выводится окно с описанием того сертификата, чей ключ совпал с {{item.name}}.
Для этого добавил тэгу <a> атрибуты class и title. Также обернул тайтл стилем посредством css (добавил св-ва в конец файла style.css)
Позиционирование пока реализовано не очень красиво, т.к. тайтл не привязан к объекту/курсору, но всегда находится в видимой области.

Для того, чтобы проверить работу на данном проекте, необходимо тэгу, в который обернута переменная item.name задать class="sert" title="".
Еще одним условием является отсутсвие конца строки в теле тэга, т.е. если написать таким образом:

<a class="sert" title="">
	{{ item.name }}
</a>,

то результатом сравнения всегда будет false. Для получения true необходимо записывать данный кусок кода так:

<a class="sert" title="">{{ item.name }}</a>

Также необходимо отредактировать title.json. Параметр key - это имя сертификата, т.е. item.name. Параметр title - это сам тайтл.
